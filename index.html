HTML

Basic Tags

Tag Name Description
< body > Body Content of the document (parent)
< p > Paragraph A basic paragraph
< div > Division Generic container for flow content, for grouping (does nothing itself)
< h1 > Heading Type of headings; < h1 > through < h6 >
< span > Content Span Create container for inline texts
< em > Emphasis Makes texts italic
< strong > Strong Importance Makes texts bold
< br > Line Break Creates a line break
< ul > Unordered List Creates an unordered list
< ol > Ordered List Creates an ordered list which is numbered
< img > Image Adds images, is self closing (only needs one tag)
< video > Video Adds videos, needs to have width and height (also add 'controls')
< audio > Audio Adds audio (must include < source > element)
< a > Anchor Together with href(attribute), creates a hyperlink to a webpage (URL)
< header > Header A container usually for navigation links or headings
< nav > Nav. Section Used to define navigation links (menus/table of contents)
< main > Main Main content of webpage (excludes < footer > & < nav >)
< footer > Footer The footer of the page
< section > Section Chapters, headings, etc
< article > Article Artcles, blogs, comments, magazines, etc
< aside > Aside Additional info (bibliographies, end notes, comments, pull notes, etc...)
< figure > Figure Images, diagrams, code snippets, etc
< figcaption > Figure Caption Adds captions or description to image or media
< embed > Embed Any media type (self closing .../>)
< style > Style Allows you to a section to write CSS code (aka CSS mode)

Table Tags

Tag Name Description
< table > Table Creates a table to display/store data
< tr > Table Row Creates a row in a table
< td > Table Data Creates table data (each cell element needs to be defined)
< th > Table Heading Creates table heading (title)
< tbody > Table Body Contains all the table data (exluding < thead >)
< thead > Table Heading The table headings for the whole table (ONLY COLUMNS)
< tfoot > Table Footer Footer of the table

Attributes for HTML

Attribute Name Description
id ID It's commonly used to name < div > tags
alt Alternative Text It's used for alternative text like a description
class Class Similar to id, its for classification purposes
target Target Specifies how a link should open (like a new tab or window)
href Href It creates links that require a URL or link to other page
colspan Column Span It allows data to span multiple columns
rowspan Row Span It allows data to span multiple rows
autoplay Auto Play It automatically plays the video when the webpage loads
controls Controls It displays controls for audio or video
src Source It specifies a source from an external source (URL)
loop Loop It loops the video or audio

CSS

Property Name Description
border Border Changes the border of an element
border-radius Border Radius Rounds the edges of the border
padding Padding Changes the space between the border and content (padding: top right bottom left)
overflow Overflow Controls what happens to content that spills out of its box
visibility Visibility Can be set to hidden, visible(default), or collapse
position Position Can be be static(default), relative, absolute, fixed, or sticky. Needs to be followed by; top, bottom, left, or right.
z-index Z-Index Controls how far back(or forward) an element should appear. Accepts integers(numbers).
display Display Displays an element inline, inline-block, and block(default)
float Float Text wrapping around an image (left or right)
clear Clear It tells how elements behave when they bump into each other (left, right, both, or none)
hsl HSL Hue, saturation, and lighting
hsla HSLA Hue, saturation, lighting, and alpha(opacity)
font-weight Font Weight Can use bold, normal(default), lighter, bolder, or a number(between 1-1000)
font-style Font Style Can make text italic
text-transform Text Transform Can make text uppercase, lowercase, or capitalized
letter-spacing Letter Spacing Can put space between letters
word-spacing Word Spacing Can put space between words
